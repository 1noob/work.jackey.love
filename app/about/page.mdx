# FAQ

**Q: 为什么不发朋友圈了？**  
**A** 一方面是觉得没有那么多的生活可以分享了，另一方面是确实不想发了。最后一条朋友圈是在复旦复试完成后发的，几天后我就接到了招生办的调剂电话，之后脑子里每天都在重复着复试那个场景，一直在懊悔和自责：如果当时做得更好，结局会不会不一样？——「失败贯穿人生始终，这就是人生」。

**Q: 做老师/公务员会觉得可惜吗？**  
**A:** 确实是有点不得志的感觉，沉没成本真的好大。但是最近看到发哥（小广财开发者）回老家当数学老师，好像也有点释怀了，互联网卷不动了，躺平当个数学老师好像也不错？

**Q: 有打算回互联网吗？**  
**A:** 其实一直都有这个想法，无论是本科时候的竞赛水平，还是毕设的问答系统，到考研复旦工研院，我一直觉得我不止于此。
